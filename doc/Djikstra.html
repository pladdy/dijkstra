    module Djikstra

    The Djikstra module is the name space containing the below classes to be used with the Djikstra algorithm:

    Djikstra::Graph Djikstra::Node (no need to use, Graph uses it when needed)

    use ‘ri’ on those for further information (ex: ri Djikstra::Graph)

    This gem can also be used as a CLI program by calling it and providing

    a file path
a starting node within the file
an ending node within the file

    in order to return the shortest path connecting your start and end nodes.

    Example: djikstra <file> <start node> <end node>

    The Djikstra::Graph class creates a graph data structure that can be built from a file with a list of edges or even one node at a time using the provided API.

    Example graph = Djikstra::Graph.new

    graph.add_node(‘A’)

    graph.add_edge(‘A’, ‘B’, 3)

    Adding edges will automatically add nodes if the nodes don’t exist in the graph already.

    Djikstra::Node handles creating node objects that belong to graphs. Nodes are automatically created in using the graph class. Since nodes can't be created separately and added to a graph it's advised that the graph api only is used.
